Occupancy Detection Dashboard
Overview:
This project is a simple web based dashboard built with Python, HTML, and JavaScript. It displays real time and historical occupancy data for a room or building. The dashboard uses charts and sliders so users can explore how occupancy changes over time.

Features:
 - Live occupancy display

 - History chart with adjustable time window

 - Slider control for choosing the time range

 - Clean layout with separate panels for data and controls
